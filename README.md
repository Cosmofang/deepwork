# DeepWork

**Collective Intent: Rethinking Collaboration in the Age of Generative AI**

> 当 AI 已经可以直接把意图变成产物，多人的想法应该如何汇聚成一个被所有人共同拥有的版本？

Deeplumen Hackathon 2025 · Present: 4/30 周三 下午 3:00

---

## 核心命题

传统协作工具的基本单位是「产物 + 评论」。  
AI 时代的协作基本单位应该是「**意图 + 合成**」。

## Demo 场景

五名队员同时打开系统，目标是共同生成一个产品落地页。  
各自用自然语言表达意图 → 系统合成出所有人都认可的 v1 → 支持集体迭代。
